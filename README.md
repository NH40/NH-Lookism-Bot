# 🥊 LOOKISM BATTLE PLANET

Telegram-игра по мотивам манхвы **Lookism** — захватывай районы, стройся в банду, торгуй на аукционах и становись Императором.

**Бот**: [@lookism_batlle_bot](https://t.me/lookism_batlle_bot)

## 🎮 Игровые фазы

### 🔴 Банда

- Выбери город и захватывай районы
- Победи во всех районах → переход в фазу Короля

### 👑 Король

- Захватывай города фазы Короля (20 городов)
- PvP с другими королями — победи и забери их город и бизнесы
- 10 городов → переход в фазу Кулака

### ✊ Кулак

- Сражайся с 10 ботами-кулаками
- Победи 10 раз → стань Императором
- Поражение = -1 город, < 7 городов = откат на Короля

### 👁 Император

- Высшее звание, доступны обновления в разработке

## ⚙️ Система механик

### ⚔️ Атака

- Кулдаун: Банда 1 мин / Король 5 мин / Кулак 10 мин
- Навык `double_attack` — 2 атаки без КД

### 👥 Группировка (Армия)

- Вербовка статистов E–S ранга
- Тренировка: улучшение звёзд бойцов (0–5)
- Боевая мощь = мощь армии + мощь персонажей

### 🃏 Колода

- Тикеты: 25% шанс каждые 5 мин, макс 3 шт.
- Гача: 9 рангов персонажей с весами
- Каждый персонаж даёт боевую мощь

### 🏢 Бизнес

- Выбор пути: Легальный / Нелегальный / Политический
- 5 зданий на каждом пути
- Строительство требует захваченных районов
- Доход начисляется каждую минуту

### 🧠 Навыки

- **Мастерство**: Сила / Скорость / Выносливость / Техника (5 уровней)
- **Пути**: Бизнесмен / Романтик / Монстр (очки пути)
- **Ультра Инстинкт**: автоматизация действий

### 🛒 Магазин

- Очки пути, Статисты (с выбором количества), Зелья
- Зелья: баффы на боевую мощь, доход, влияние и др.

### 🔨 Аукцион

- 5 уровней редкости: Уличный / Районный / Городской / Королевский / Императорский
- Раунды: 2 / 3 / 6 / 2 / 6
- Лоты: тикеты, персонажи, статисты, зелья, очки пути
- Ставки: +5% / +10% / +50% или своя сумма
- Таймер: 60 сек + 5 сек при новой ставке
- Запуск каждые 20 минут, уведомления подписчикам

### 🔗 Реферальная система

- Уникальная реферальная ссылка для каждого игрока
- Бонус новому игроку при регистрации
- Пригласившему начисляются награды за прогресс реферала

### 🏆 Титулы

- 14 достижений с бонусами
- 5 донатных титулов (через менеджера)
- True Ультра Инстинкт — безлимитное хранилище тикетов

## 🚀 Запуск

### Требования

- Docker Desktop
- Git

### Установка

```bash
git clone https://github.com/NH40/NH-Lookism-Bot.git
cd NH-Lookism-Bot
```

Создай `.env` файл:

```env
BOT_TOKEN=your_bot_token_here

POSTGRES_USER=lookism
POSTGRES_PASSWORD=lookism_secret
POSTGRES_DB=lookism_db
POSTGRES_HOST=postgres
POSTGRES_PORT=5432

REDIS_HOST=redis
REDIS_PORT=6379
REDIS_DB=0

ADMIN_IDS=123456789,987654321

DEBUG=false
```

Запуск:

```bash
docker-compose up -d --build
```

Проверка:

```bash
docker-compose logs bot --tail=20
```

## 💾 Система бэкапов

Бэкапы создаются через `/admin` и хранятся в `/app/backups/`.

Откат к бэкапу работает через отдельный `manager` контейнер:

1. Бот записывает запрос на откат
2. Менеджер завершает активные соединения
3. Дропает и восстанавливает схему
4. Бот перезапускается автоматически
