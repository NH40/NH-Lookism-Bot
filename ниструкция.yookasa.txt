📋 Пошаговая инструкция для тебя

Шаг 1 — Регистрация в ЮKassa (20 мин)
Зайди на yookassa.ru
Нажми «Подключить» → «Физическое лицо / ИП»
Введи данные ИП: ИНН, ОГРНИП
Пройди верификацию (потребует скан ИП или ЕГРИП)
В разделе «Интеграции» — запиши shopId и secretKey (они нужны в следующем шаге)

Шаг 2 — Подключение ЮKassa к боту в @BotFather (5 мин)
Открой Telegram → найди @BotFather
Отправь /mybots → выбери своего бота
Bot Settings → Payments
Выбери «ЮKassa»
BotFather спросит shopId и secretKey из шага 1
Введи их — получишь payment provider token вида 1234567890:LIVE:xxxxxxxxxxxxxx
Скопируй этот токен

Шаг 3 — Добавить токен в .env (2 мин)
Открой файл .env в корне проекта и добавь строку:


YOOKASSA_PAYMENT_TOKEN=1234567890:LIVE:xxxxxxxxxxxxxx
Шаг 4 — Применить миграцию БД (2 мин)

docker exec -it lookism_postgres psql -U lookism -d lookism_db -c "
ALTER TABLE users ADD COLUMN IF NOT EXISTS nh_donate INTEGER DEFAULT 0;
CREATE TABLE IF NOT EXISTS payments (
    id SERIAL PRIMARY KEY,
    user_id INTEGER NOT NULL,
    tg_payment_charge_id VARCHAR(128),
    provider_payment_charge_id VARCHAR(128),
    amount_rub INTEGER NOT NULL,
    nh_donate_credited INTEGER NOT NULL,
    payload VARCHAR(256),
    status VARCHAR(16) NOT NULL DEFAULT 'success',
    created_at TIMESTAMPTZ DEFAULT NOW()
);
CREATE INDEX IF NOT EXISTS idx_payments_user ON payments(user_id);
"

Шаг 5 — Перезапуск бота (1 мин)

docker-compose up -d --build
docker-compose logs bot --tail=30
Шаг 6 — Тест
Напиши боту /donate
Выбери сумму, например 50 ₽
Появится счёт Telegram — введи тестовую карту 1111 1111 1111 1026 (ЮKassa тест)
После оплаты придёт сообщение ✅ Оплата прошла успешно! +50 NHDonate
⚠️ Пока YooKassa в тестовом режиме (до прохождения верификации), используй тестовые карты. После одобрения ИП аккаунта — боевые платежи.
